# Backend

## Structure

- `src/`
  - `controllers/`
  - `models/`
  - `routes/`
  - `services/`
  - `middleware/`
  - `config/`
- `tests/`
- `package.json`
- `server.js`

## Files
- **controllers/**: contains controllers to handle requests
- **models/**: defines database models
- **routes/**: sets up API routes
- **services/**: contains business logic
- **middleware/**: middleware functions
- **config/**: configuration files
